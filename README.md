# bibliotheca-pwa

## Requirements

- Node.js 8.x (Cloud Functions for Firebase のため)

## Development

どのパッケージで開発する場合も、プロジェクトルートで `yarn tsc-w` を実行すること

## Project の切り替え

```bash
$ firebase use test
$ firebase use prod
```
